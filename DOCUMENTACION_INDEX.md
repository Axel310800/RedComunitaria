## 📞 Resumen Rápido de Procesos

### Iniciar Frontend
```bash
pnpm install
pnpm run dev
# http://localhost:4200
```

### Iniciar Backend
```bash
cd backend
venv\Scripts\activate.bat
python src/main.py
# http://localhost:8000
```

### Crear BD
```bash
mysql -u root -p < scripts/init_db.sql
mysql -u root -p < scripts/seed_db.sql
```
